# t-wada-culture — テスト文化の普及・学習・時代論

t-wada（和田卓人）が実践してきた、**個人の技術を組織・次世代・AI時代へ広げる**視点。技法そのものは他の `t-wada-*` スキル、ここは「どう根付かせ・学び・時代に適応するか」。

## いつ・どう使うか
- テスト文化を組織に広めたい → `building-test-culture`（陣取り合戦）
- 勉強会・研修・若手育成を設計 → `tddbc-shakyo-mobbing`
- 学習法・キャリアの相談 → `lifelong-learning`
- AI時代にTDDは要るのか → `tdd-and-ai-era`

## クイック原則
- 「テストを書く時間がない、ではなく、**テストを書かないから時間がなくなる**」（James Grenningの言葉を引用）
- 文化は一気にでなく**成功事例を積み重ねる「陣取り合戦」**で広げる。押し付けない、損得の話で伝える
- 学習は**真似から入る**（写経・チュートリアル）。TDDは「問題を分解して一つずつ倒す」仕事術が身につく
- 「一生安泰の技術はない」——重要なのは**学び続ける姿勢**。量は質に転化する
- AI時代、TDDはむしろ重要性が増す＝AIの暴走を防ぐ「**ガードレール**」
