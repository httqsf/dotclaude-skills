# lifelong-learning — 一生勉強・審美眼・AI時代の生存戦略

## 核心
「これを学べば一生安泰」という技術は存在しない。だからこそ本当に重要なのは特定技術ではなく「**学ぶスキル・学び続ける姿勢**」そのもの。長く深く続けることで技術は身体の一部になり、**量稽古＝量は質（＝設計の判断力）に転化**する。キャリア初期から柱を1本に絞らず、10年ほどかけて3つほどの得意分野（柱）を持つのが破壊的技術に対して安全。

## 指針
- **Write Code Every Day**：毎日コードを書く（ブログ・ドキュメントも「書く」に含む。整形やリファクタのみはカウントしない）。文脈切替コストが減り、常時コードを考える状態になる。
- **四半期に1つ新しい言語を学ぶ**（少なくとも年1言語）。
- 技術的審美眼は「**答え合わせ**」で磨く：どの技術が定着しどれが廃れたかを検証し、選択と結果のレビューを繰り返して打率を上げる。
- ゲームチェンジャー（インターネット、クラウド、LLM等の避けられない波）には積極的に乗る。それ以外は概要を掴んで静観。
- ベテランは「過去の経験からの類推」で情報を取捨選択し、必要に応じて**アンラーン（学びほぐし）**する。
- 「小さいものを企画から運用まで全部1人で作る経験」で全体像とトレードオフ感覚を養う。
- 学習動機はコスパ・タイパでは続かない。「**好きでのめり込む**」こと。

## AI時代の生存戦略
- 生き残るスキルは「**構造化と言語化**」——問題を正しく構造化し、人にもコンピュータにも正確に伝える力。
- 「リソース効率」から「**フロー効率**」へのシフト。AIの高速生成に対し人間は即断即決できるチーム体制で応じる。
